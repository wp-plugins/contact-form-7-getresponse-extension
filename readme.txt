A very easy plugin to integrate GetResponse campaigns with Contact Form 7.
== Description ==
Contact Form 7 GetResponse extension is a simple yet useful plug-in to integrate Contact Form 7 with GetResponse. Using CF7 form, contacts can be automatically added to GetResponse campaigns. Different campaigns can be used on different CF7 forms. It also supports GetResponse custom fields which helps to collect required data from the users.

= Feature Include =
* Simplicity
* Different Campaigns for different forms
* GetResponse Custom fields supported
* Opt-in checkbox, which gives user option to subscribe or not

= Requirements =
* PHP 5.3 +
* Self hosted WordPress.org installation (3.9 + )
* Contact Form 7 (4.2 +)
* Account on GetResponse

== Installation ==
* Upload the entire 'contact-form-7-getresponse-extension' folder to the '/wp-content/plugins/' directory.
* Activate the plugin through the 'Plugins' menu in WordPress.
For more detail [Contact Form 7 GetResponse Extension page](http://racase.com/contact-form-7-getresponse-extension).
